# Infra Reviewer

🔵 Freemium · 🚧 WIP

A Claude Agent SDK agent that **autonomously reviews Terraform and Docker changes** and comments on the PR — security misconfigs, cost spikes, drift risks, missing tags.

## Who it's for

DevOps / platform teams who want a second set of eyes on every infra change.

## On-thesis

Makes infra **safer and cheaper** by catching problems at review time. Free to run locally; paid as a hosted GitHub app.
