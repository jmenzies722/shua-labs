# /threat-model

🟢 Free · 🚧 WIP

A Claude Code skill that points at a repo and produces an **OWASP-style threat model** — entry points, trust boundaries, likely attack paths, and mitigations.

## Who it's for

Builders who want a fast, structured security pass without a security hire.

## On-thesis

Makes agents/systems **safer** — surfaces risk before it ships.
